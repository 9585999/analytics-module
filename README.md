## Setup
- Add `@nuxtjs/google-analytics` dependency using yarn or npm to your project
- Add `@nuxtjs/google-analytics` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    '@nuxtjs/google-analytics',

    // With options
    ['@nuxtjs/google-analytics', { /* module options */ }],
 ]
}
```
